Find Largest Sum Contiguous Subarray

The problem of identifying the contiguous portion of an array with the highest sum of its elements.
For example, in the array [-2, 1, -3, 4, -1, 2, 1, -5, 4], the subarray [4, -1, 2, 1] has the largest sum of 6.
This problem can be efficiently solved using Kadane's Algorithm.
